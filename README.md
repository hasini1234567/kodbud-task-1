Introduction

Data analysis plays a vital role in understanding the spread and impact of infectious diseases. In this project, COVID-19 data is analyzed to study daily confirmed cases, deaths, and recovery trends. The analysis helps identify patterns, monitor disease progression, and understand the overall impact of the pandemic through data visualization techniques.

Objective

The main objectives of this project are:

* Analyze daily COVID-19 confirmed cases.
* Study death trends over time.
* Examine recovery patterns.
* Compare confirmed, recovered, and death cases.
* Identify countries or regions with the highest impact.
* Visualize COVID-19 trends using charts and graphs.
* Learn data cleaning, analysis, and visualization techniques.

Dataset Description

The dataset contains COVID-19 information with the following attributes:

* Date
* Country/Region
* Province/State
* Confirmed Cases
* Deaths
* Recovered Cases
* Active Cases
* Latitude
* Longitude

Data Preprocessing

The following preprocessing steps were performed:

1. Downloaded the COVID-19 dataset from Kaggle.
2. Loaded the dataset using Pandas.
3. Checked for missing values.
4. Removed duplicate records.
5. Converted date columns into proper datetime format.
6. Verified data types.
7. Cleaned and organized the dataset.
8. Prepared data for analysis and visualization.

 Exploratory Data Analysis
